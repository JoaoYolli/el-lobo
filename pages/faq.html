Preguntas Frecuentes (FAQ)

¿Qué es "Los Lobos de Castronegro"?

"Los Lobos de Castronegro" es un juego de roles donde los jugadores asumen el papel de aldeanos y lobos. Los lobos deben eliminar a los aldeanos sin ser descubiertos, mientras que los aldeanos intentan identificar y eliminar a los lobos. Nuestro sitio web actúa como un narrador automático para facilitar el juego entre amigos.

¿Cómo se juega a "Los Lobos de Castronegro" usando este narrador?

Para jugar usando nuestro narrador:

  1. Crea una sala y configura el juego.
  2. Comparte el código de la sala con los jugadores.
  3. Sigue las instrucciones del narrador para llevar a cabo las fases del juego.

¿Cuántos jugadores pueden participar en una partida?

El juego es ideal para un grupo de 4 a 20 jugadores. Esto asegura que haya suficientes roles para hacer el juego interesante y desafiante. Sin embargo puedes jugar con el grupo que quieras ya que no esta limitado, solo recibiras una recomendacion y podrás continuar.

¿El narrador se puede personalizar?

Sí, puedes personalizar los roles de la partida antes de jugar. También puedes ajustar la duración de las fases para adaptar el juego a las preferencias de tu grupo.

¿Es necesario registrarse para jugar?

Es necesario para unirse a una partida, sin embargo no lo es para hostear un juego. Esto se hace para poder tener un registro de las estadisticas de cada jugador y poder tener un ranking.

¿Es gratis usar el narrador de "Los Lobos de Castronegro"?

Sí, el uso de nuestro narrador es completamente gratuito. No hay versiones u opciones de pago.
